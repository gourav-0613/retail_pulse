RetailPulse is an end-to-end Machine Learning and Business Analytics project that helps retail businesses analyze historical sales data, forecast future demand, identify unusual sales patterns, and segment products based on demand.

The application is deployed as an interactive **Streamlit Dashboard** where users can explore business insights without writing code.
